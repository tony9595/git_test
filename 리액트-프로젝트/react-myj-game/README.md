3. 프리티어 설정
{

}

Path Alias with Vite

jsonconfig.json 파일을 만든후
아래를 추가

```
{
    "compilerOptions": {
    "baseUrl": ".",
    "paths": {
    "@/": ["./src/_"]
    }
    },
    "exclude": ["node_modules"] //프로젝트 규모가 크다면 성능(intellisense)이 낮아질 수 있는데 이 때 내가 실질적으로 사용하지 않는 파일들을 제외해줄 수 있다.
}
```
